# Портативный тепличный комплекс с полным контролем микроклимата

**Ментор**: Павел Владимирович
**Команда**: -

# 1. Аннотация

Проект направлен на разработку портативного тепличного комплекса, обеспечивающего автоматизированный контроль и регулирование агротехнических параметров микроклимата системы: влажность, температура, освещенность и влажность почвы.

# 2. Цель проекта

Разработка портативного тепличного комплекса с замкнутым контуром управления микроклиматом, обеспечивающего следующие возможности:
- автоматическое поддержание заданной влажности и температуры воздуха
- циклическое увлажнение почвы до необходимой влажности
- минимальное участие пользователя в работе системы
- настройка параметров комплекса для различных агротехнических парамеров растений через веб-интерфейс

# 3. Технические требования
## 3.1 Диапазоны регулирования
| Параметр                | Диапазон                                   |
| ----------------------- | ------------------------------------------ |
| Влажность воздуха       | от комнатной (~30–40%) до 95%              |
| Температура воздуха     | +15…+40 °C                                 |
| Освещённость            | регулируемая (искусственное освещение)     |
| Влажность почвы         | автоматическое поддержание                 |
| Время автономной работы | ≥ 72 часа (без вмешательства пользователя) |

## 3.2 Габариты

# Описание функционала

Контроль влажности воздуха в пределах [Комнатная влажность, 95%]
Контроль температуры воздуха в пределах [15-40 градусов по Цельсию]
Автономный автоматический полив до 3 суток
Размеры тепличного комплекса составляют до 800 * 800 * 1000 мм.

# Задачи проекта

- Выбор доступной элементной базы
- Разработка способов контроля температуры и влажности воздуха
- Поиск возможностей организации автоматического полива растений
- Создание CAD моделей деталей проекта
- Разработка схем электрических компонентов
- Проведение CAE расчетов
- Производство деталей прототипа
- Сборка и тестирования устройства
