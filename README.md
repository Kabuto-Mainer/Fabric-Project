# 1. Аннотация

Проект направлен на разработку портативного тепличного комплекса, обеспечивающего автоматизированный контроль и регулирование агротехнических параметров микроклимата системы: влажность, температура, освещенность и влажность почвы.

# 2. Цель проекта

Разработка портативного тепличного комплекса с замкнутым контуром управления микроклиматом, обеспечивающего следующие возможности:
- автоматическое поддержание заданной влажности и температуры воздуха
- циклическое увлажнение почвы до необходимой влажности
- минимальное участие пользователя в работе системы
- настройка параметров комплекса для различных агротехнических парамеров растений через веб-интерфейс

# 3. Технические требования

## 3.1 Диапазоны регулирования

| Параметр                | Диапазон                                   |
| ----------------------- | ------------------------------------------ |
| Влажность воздуха       | от комнатной (~30–40%) до 95%              |
| Температура воздуха     | +15…+40 °C                                 |
| Освещённость            | регулируемая (искусственное освещение)     |
| Влажность почвы         | автоматическое поддержание                 |
| Время автономной работы | ≥ 72 часа (без вмешательства пользователя) |

## 3.2 Габариты комплекса
- Максимальные размеры установки: 800 * 800 * 1000 мм
- Тип конструкции: Полугерметичное основание и герметичный внешний корпус
- Материал: Фанера или пластик для основания и акрил для корпуса

# 4. Функционал системы
Система должна обеспечивать:

## 4.1. Контроль параметров среды

- измерение температуры воздуха;
- измерение относительной влажности воздуха;
- измерение освещённости;
- измерение влажности почвы;
- контроль уровня воды в резервуаре.

## 4.2. Регулирование

- управление системой увлажнения воздуха;
- управление вентиляцией (приток/вытяжка);
- управление нагревом (при необходимости);
- управление освещением;
- автоматический полив растений.

## 4.3. Автоматизация

- работа по заданным уставкам;
- возможность настройки пользователем;
- поддержание параметров по обратной связи (замкнутый контур управления).


# 5. Задачи проекта

- Анализ и выбор элементной базы;
- Разработка алгоритмов управления микроклиматом;
- Проектирование электрической схемы;
- Разработка CAD-модели конструкции;
- Проведение CAE-расчётов (тепловые потоки, циркуляция воздуха);
- Изготовление прототипа;
- Сборка системы;
- Проведение испытаний и калибровка.

# 6. Элементная база (предварительная)

## 6.1. Контроллер

- Arduino Mini / Arduino Nano / ESP32 (предпочтительно для расширения функционала)

## 6.2. Сенсоры

- Датчик температуры и влажности воздуха (например: DHT22 / SHT31)
- Датчик освещённости (LDR / BH1750)
- Датчик влажности почвы (ёмкостной тип)
- Датчик уровня воды

## 6.3. Исполнительные устройства

- Вентиляторы (DC)
- Насос (перистальтический или погружной)
- Увлажнитель (ультразвуковой модуль или пуливеризатор)
- Светодиодные модули (LED grow light)

## 6.4. Дополнительные компоненты

- Воздушный фильтр (для приточного воздуха)
- Резервуар для воды
- Воздуховоды / каналы
- Корпус (акрил/поликарбонат)
